# UoG_TDP-EE3_Team_7
Design Files for the 3rd year EEE design project at the University of Glasgow. Line following rover, with obstacle avoidance, traffic lights detection, remote control using an ESP-32 connected to a laptop and a Logitech 3D Pro Joystick.
The  Centeral processor is a Freescale NXP Frdm KL25Z.
